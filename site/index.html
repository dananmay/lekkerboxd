Better Letterboxd picks, in one click.

Lekkerboxd scans your public Letterboxd activity and generates recommendations using TMDb, Reddit, and Taste.io signals. Fast results, background generation, and clean filtering.

Install on Chrome View Source

No account creation required. Works with your existing Letterboxd login.

Store build: no TMDb key required Pop-up + pop-out modes One-click watchlist add

Signal-aware recommendations

Combines TMDb graph data with Reddit and Taste.io signals, then ranks with quality filters.

Built for browsing flow

Runs in the popup, can pop out to its own window, and keeps generation running in the background.

User-first controls

Customize seeds, recommendation count, popularity strictness, and JustWatch region routing.

Get started in under a minute

1

Install Lekkerboxd from the Chrome Web Store and pin it from the puzzle menu.

2

Open Letterboxd while logged in and click Scan Profile.

3

Click Get Recommendations and start exploring titles instantly.
